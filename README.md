# react-native-vibration-effect

Android's VibrationEffect for React Native

## Installation

```sh
npm install react-native-vibration-effect
```

## Usage

```js
import { multiply } from "react-native-vibration-effect";

// ...

const result = await multiply(3, 7);
```
